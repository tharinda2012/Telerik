Autogen v0.1

This tool can be used to create the object repository class that we used in UI tests. It reads data from excel file and create a text file. 

Follow these instructions before using this tool

1. Install Python 3 or above in your machine

2. Try to run the auto_gen.py first time, if the "xlrd" module is not available message appears, install it using following method
       1. Go to Python %installation_folder%/Lib/site_packages using windows command prompt
       2. type: easy_install.py xlrd   

3. Keep both auto_gen.py and element_file.xlsx in same folder location
